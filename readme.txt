WordPress plugin to be used with Woocommerce this will add an extra custom field to your products with ease.

== Description ==

> Easily add a custom field to your woocommerce products this plugin wil do just that.

= Overview =
Adding a custom EAN field to your woocommerce products, will help you create your Google Merchant feed. This plugin will not create the feed for you, but the custom field is mandatory.
There is no admin/settings screen.

= Installation Instructions =
1. Upload `s2 Woocommerce EAN` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
